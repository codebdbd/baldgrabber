# BaldGrabber

Download videos and audio from YouTube on Windows.

License: [MIT](LICENSE)

## Features

- Download videos as MP4 (240p to 4K)
- Download audio as Opus, M4A, MP3 (96/128 kbps)
- Download playlists and channels
- Trim audio and video by timestamps
- Automatic cover art embedding in audio files
- Display download speed and remaining time
- Auto-detect available formats for each video
- Localization: Russian, English, Ukrainian

## System Requirements

- Windows 10 version 1809 (build 17763) or newer
- 64-bit system (x64)
- ~50 MB of free disk space
- .NET 9 Runtime (included in self-contained build)

## Installation

Download `BaldGrabber_Installer.exe` and run it. The app will install to `C:\Program Files\BaldGrabber` and create a desktop shortcut.

A portable version is also available — extract the archive to any folder and run `BaldGrabber.exe` directly.

## Quick Start

1. Paste a YouTube video link
2. Choose a save folder
3. Select mode (Audio / Video) and quality
4. Click "Download"

## Supported Formats

| Mode | Format | Description |
|------|--------|-------------|
| Video | MP4 | 240p — 4K, merged video + audio |
| Audio | Opus | YouTube's native codec |
| Audio | M4A | Lossless AAC |
| Audio | MP3 128 | Universal, small size |
| Audio | MP3 96 | Minimal size, for podcasts |

# BaldGrabber

Скачивание видео и аудио с YouTube на Windows.

Лицензия: [MIT](LICENSE)

## Возможности

- Скачивание видео в MP4 (от 240p до 4K)
- Скачивание аудио в Opus, M4A, MP3 (96/128 kbps)
- Скачивание плейлистов и каналов
- Обрезка аудио и видео по временным меткам
- Автоматическая вставка обложки в аудиофайлы
- Отображение скорости загрузки и оставшегося времени
- Автоопределение доступных форматов для конкретного видео
- Локализация: русский, английский, украинский

## Системные требования

- Windows 10 версии 1809 (сборка 17763) или новее
- 64-разрядная система (x64)
- ~50 МБ свободного места на диске
- .NET 9 Runtime (входит в self-contained сборку)

## Установка

Скачайте установщик `BaldGrabber_Installer.exe` и запустите его. Программа установится в `C:\Program Files\BaldGrabber` и создаст ярлык на рабочем столе.

Также доступна портативная версия — распакуйте архив в любую папку и запустите `BaldGrabber.exe` напрямую.

## Краткое использование

1. Вставьте ссылку на YouTube-видео
2. Выберите папку для сохранения
3. Выберите режим (Аудио / Видео) и качество
4. Нажмите «Скачать»

## Поддерживаемые форматы

| Режим | Формат | Описание |
|-------|--------|----------|
| Видео | MP4 | 240p — 4K, объединение видео + аудио |
| Аудио | Opus | Оригинальный кодек YouTube |
| Аудио | M4A | AAC без потерь |
| Аудио | MP3 128 | Универсальный, малый размер |
| Аудио | MP3 96 | Минимальный размер, для подкастов |
