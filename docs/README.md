# BaldGrabber

Скачивание видео и аудио с YouTube на Windows.

## Возможности

- Скачивание видео в MP4 (от 240p до 4K)
- Скачивание аудио в Opus, M4A, MP3 (96/128 kbps)
- Скачивание плейлистов
- Обрезка аудио и видео по временным меткам
- Автоматическая вставка обложки в аудиофайлы
- Отображение скорости загрузки и оставшегося времени
- Автоопределение доступных форматов для конкретного видео
- Локализация: русский, английский, украинский

## Системные требования

- Windows 10 версии 1809 (сборка 17763) или новее
- 64-разрядная система (x64)
- ~50 МБ свободного места на диске
- .NET 9 Runtime (входит в self-contained сборку)

## Установка

Скачайте установщик `BaldGrabber_Installer.exe` и запустите его. Программа установится в `C:\Program Files\BaldGrabber` и создаст ярлык на рабочем столе.

Также доступна портативная версия — распакуйте архив в любую папку и запустите `BaldGrabber.exe` напрямую.

## Краткое использование

1. Вставьте ссылку на YouTube-видео
2. Выберите папку для сохранения
3. Выберите режим (Аудио / Видео) и качество
4. Нажмите «Скачать»

## Поддерживаемые форматы

| Режим | Формат | Описание |
|-------|--------|----------|
| Видео | MP4 | 240p — 4K, объединение видео + аудио |
| Аудио | Opus | Оригинальный кодек YouTube |
| Аудио | M4A | AAC без потерь |
| Аудио | MP3 128 | Универсальный, малый размер |
| Аудио | MP3 96 | Минимальный размер, для подкастов |
